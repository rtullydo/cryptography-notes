Definition 3.5.1.

Let \(G\) be a set of elements, and let \(\star: G \times G \to G\) be an operation that combines two elements of \(G\) into a single element in \(G\text{.}\) We say that \(G\) is a group under \(\star\) if the operation \(\star\) has the following three properties.

  1. Identity: there exists an element \(e\in G\) such that for all \(g \in G\text{,}\) we have \[ g\star e = e \star g = g. \]
  2. Inverses: for all \(g \in G\text{,}\) there exists an element \(g\inv \in G\) so that \[ g \star g\inv = g\inv \star g = e. \]
  3. Associativity: For all \(a, b, c \in G\text{,}\) we have \[ a \star (b \star c) = (a \star b) \star c. \]
If in addition it is the case that for all \(a, b \in G\) we have \[ a \star b = b \star a \] then \(G\) is called a commutative group.
