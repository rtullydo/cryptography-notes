Suppose that we apply \(f\) a total of \(n\) times to the starting point \(x_0 = 1\text{.}\) Then

\begin{equation*} f\circ f \circ f \circ \ldots \circ f \circ f (1) = g^{a_n} h^{b_n} \end{equation*}

We can generate the “storage constants” as we calculate the list with the conditions

\begin{equation} a_n = \left\{ \begin{array}{lll} a_{n-1} +1 \amp \mbox{if } \amp 1 \leq x \lt p/3 \\ 2a_{n-1} \amp \mbox{if } \amp p/3 \leq x \lt 2p/3 \\ a_{n-1} \amp \mbox{if } \amp 2p/3 \leq x \lt p \\ \end{array} \right.\tag{6.4.2} \end{equation}

and

\begin{equation} b_n = \left\{ \begin{array}{lll} b_{n-1} \amp \mbox{if } \amp 1 \leq x \lt p/3 \\ 2b_{n-1} \amp \mbox{if } \amp p/3 \leq x \lt 2p/3 \\ b_{n-1} +1 \amp \mbox{if } \amp 2p/3 \leq x \lt p \\ \end{array} \right.\tag{6.4.3} \end{equation}
