Theorem 4.2.2. Elliptic curve addition.
\(E\)\(P, Q\)\(E\text{.}\)
  1. If \(P = \mathcal O\text{,}\) then \(P+ Q = Q\text{.}\)
  2. Else, if \(Q = \mathcal O\text{,}\) then \(P + Q = P\text{.}\)
  3. Else, write the coordinates \(P = (x_1, y_1), Q = (x_2, y_2)\text{.}\)
  4. If \(x_1 = x_2\) and \(y_1 = -y_2\) then \(P+ Q = \mathcal O\text{.}\)

  5. Else, compute the slope \(m\text{.}\)

    1. If \(P \neq Q\) then
      \begin{equation*} m = \frac{y_2 - y_1}{x_2 - x_1}. \end{equation*}
    2. Else
      \begin{equation*} m = \frac{3x_1^2 + A}{2y_1}. \end{equation*}
  6. Then
    \begin{gather*} x_3 = m^2 - x_1 - x_2,\\ y_3 = m(x_1 - x_3) - y_1. \end{gather*}
    and \(P + Q = (x_3, y_3).\)
