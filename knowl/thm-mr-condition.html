Theorem 6.3.3.
Let \(p\) be an odd prime. Write
\begin{equation*} p - 1 = 2^k q \text{ where } q \text{ is odd.} \end{equation*}
Let \(a\) be any number so that \(p \nmid a\text{.}\) One of the following must be true:
  1. \(a^q \equiv 1 \bmod p;\)
  2. A number in the list
    \begin{equation} a^q, a^{2q}, a^{4q} \ldots, a^{2^{k-1}q}\tag{6.3.1} \end{equation}
    is congruent to \(-1 \bmod p\text{.}\)
